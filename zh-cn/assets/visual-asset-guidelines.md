# Visual Asset Guidelines

## Media System

```yaml
media_system:
  image_style:
    subject_style: "real product, person, object, interface, or content state that supports the page goal"
    composition: "clear subject, generous margins, stable crop, no essential text baked into the image"
    crop_ratio: "16:9 for media grids, 4:3 for editorial cards, 1:1 for avatars or products, 3:4 for portrait features"
    lighting: "soft and readable with enough contrast for subject recognition"
    texture: "subtle, never noisy behind text"
    depth: "shallow to moderate depth for feature imagery, flat for interface captures"
  illustration_style:
    geometry: "simple geometric shapes with replaceable motifs"
    line_weight: "1.5px to 2px equivalent"
    fill_strategy: "neutral fills with restrained brand and accent highlights"
    metaphor_style: "clear task metaphors, no proprietary characters or symbols"
  background_treatment:
    gradient: "subtle surface wash only when it improves hierarchy"
    pattern: "low-contrast abstract pattern, optional"
    noise: "very subtle, avoid text areas"
    blur: "only for overlay depth or non-essential ambience"
    decorative_shapes: "minimal, generic, and never brand-locked"
```

## Replacement Rules

- Do not use protected images, logos, trademarks, characters, proprietary symbols, or distinctive artwork.
- Preserve only role, composition, ratio, density, tone, and semantic purpose.
- Replace all visual subjects with new assets that fit the new brand and content.
- Avoid embedding critical text in imagery.
- Ensure media has alt text rules appropriate to its purpose.
