# 响应式清单

## Desktop

- 内容适配最大容器宽度。
- 网格列保持卡片尺寸稳定。
- 工具导航不遮挡内容。
- 侧边面板不会压缩主内容到不可读。
- hover 状态不造成布局位移。

## Tablet

- 网格根据最小卡片宽度减少为 2 或 3 列。
- 侧边导航可折叠或保持可读。
- 点击区域达到最小尺寸。
- 表单和搜索保持可用。

## Mobile

- 导航移动到顶部行、底部栏、drawer 或 sheet。
- 主内容只在卡片仍可读时使用 1 或 2 列。
- 文本不溢出按钮、卡片、输入框或标签。
- 媒体保持稳定比例。
- modal 和 drawer 适配视口高度。
- 无水平滚动。

## State Coverage

- loading 状态保持组件尺寸。
- error 和 empty 状态适配同一响应式外壳。
- focus indicator 在视口边缘仍然可见。
- sticky 元素不遮挡标题、输入或操作。
