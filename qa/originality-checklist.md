# 原创性清单

## Required Checks

- 无品牌名、产品名、路由标签、受保护标记或独特短语。
- 无受保护图片、截图、标志、mascot、专有图标或独特 artwork。
- 无现有产品的独特 CSS 选择器、组件名、文件名或导航用语。
- 无只适合单一页面的像素级锁定布局值。
- 色彩已转化为语义化 palette role。
- 布局已转化为灵活网格和外壳规则。
- 组件已转化为带可替换 anatomy 的通用 schema。
- 文案已转化为内容角色。
- 媒体已转化为构图与替换指南。
- 交互已转化为行为模式。
- 品牌表达已转化为可调节维度。

## Pass Criteria

- 该包可以作为新的设计系统独立使用。
- 该包可以生成多个网站类型。
- 该包不包含外部品牌痕迹。
- 替换颜色、内容、图片和品牌语气后仍然有用。
