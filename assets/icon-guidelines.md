# 图标指南

## Icon System

```yaml
icon_system:
  stroke_width: "1.75px to 2px"
  corner_style: "rounded joins and caps"
  filled_or_outline: "默认 outline，只在 selected 或 status 强调时 filled"
  size_scale:
    sm: "16px"
    md: "20px"
    lg: "24px"
```

## Rules

- 图标用于重复操作、工具、筛选和紧凑导航。
- 不熟悉图标需要标签或 tooltip。
- icon-only 控件桌面至少 `40px`，移动端至少 `44px`。
- selected 图标可使用 filled 处理，并搭配文字或 active marker。
- 状态图标必须搭配文字，不只靠颜色。
- 一个界面只使用一个图标体系。
- 避免专有符号和品牌标志。

## Common Categories

- Navigation：home、search、categories、saved、history、settings、account。
- Content：article、media、product、document、collection、upload。
- Feedback：info、success、warning、danger、loading。
- Commerce：cart、favorite、compare、shipping、payment。
- Documentation：code、copy action、external link、previous、next。
