# 视觉资产指南

## Media System

```yaml
media_system:
  image_style:
    subject_style: "支持页面目标的真实产品、人物、物体、界面或内容状态"
    composition: "主体清楚、留白充分、裁切稳定、图片中不嵌入关键信息文字"
    crop_ratio: "媒体网格 16:9，编辑卡片 4:3，头像或商品 1:1，人物特写 3:4"
    lighting: "柔和且可读，主体识别有足够对比"
    texture: "细腻，不在文字区域制造噪音"
    depth: "feature imagery 使用浅到中等景深，interface capture 保持平面"
  illustration_style:
    geometry: "简单几何形状和可替换 motif"
    line_weight: "等效 1.5px 到 2px"
    fill_strategy: "中性填充，品牌色和强调色克制点缀"
    metaphor_style: "清楚任务隐喻，不使用专有角色或符号"
  background_treatment:
    gradient: "只在改善层级时使用轻微表面渐变"
    pattern: "低对比抽象 pattern，可选"
    noise: "非常轻微，避开文字区域"
    blur: "只用于浮层深度或非必要氛围"
    decorative_shapes: "最少、通用、不绑定品牌"
```

## Replacement Rules

- 不使用受保护图片、标志、商标、角色、专有符号或独特 artwork。
- 只保留角色、构图、比例、密度、调性和语义用途。
- 所有视觉主体都替换为适合新品牌和内容的新资产。
- 避免在图片中嵌入关键信息文字。
- 媒体根据用途提供 alt text 规则。
