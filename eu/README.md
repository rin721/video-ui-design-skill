# UI Design Skill Package

This package contains an original, reusable UI design skill for generating modern website systems. It includes semantic tokens, component specifications, page templates, visual asset rules, prompt blocks, and QA checklists.

## Contents

- `SKILL.md`: main skill entry and operating contract.
- `design-system/`: principles, tokens, layout, typography, color, accessibility, and brand expression.
- `components/`: reusable component schemas and behavior rules.
- `patterns/`: adaptable page templates for common site types.
- `prompts/`: standalone generation and review prompt blocks.
- `assets/`: visual, icon, and media replacement guidance.
- `qa/`: originality, accessibility, responsive, and cleanup checklists.

## Ready For

- Website generation.
- Style swapping.
- Component generation.
- Page template generation.
- Brand adaptation.
- Accessibility review.

## Core Style

The system favors airy surfaces, compact utility navigation, soft accent emphasis, clear content hierarchy, restrained elevation, and dense but readable content grids. All rules are expressed as reusable variables, patterns, and component contracts.
