# Brand Expression

## Brand Grammar

```yaml
brand_expression:
  personality:
    - "approachable"
    - "fresh"
    - "organized"
    - "lightly playful"
    - "trustworthy"
  emotional_goal:
    - "make dense information feel friendly"
    - "reduce friction in discovery and decision making"
    - "invite repeated engagement without visual fatigue"
  trust_strategy:
    - "clear hierarchy"
    - "predictable controls"
    - "legible metadata"
    - "consistent feedback"
  differentiation_strategy:
    - "soft accent system"
    - "compact navigation"
    - "balanced content density"
    - "replaceable visual rhythm"
  voice_visual_alignment:
    - "short confident labels"
    - "warm helper text"
    - "precise status wording"
    - "direct calls to action"
  user_expectation:
    - "quick orientation"
    - "fast scanning"
    - "low-friction interaction"
    - "safe recovery from errors"
  brand_swap_parameters:
    seriousness: "medium"
    warmth: "medium"
    innovation: "medium"
    luxury: "low"
    playfulness: "medium"
    editorial: "medium"
    technical_depth: "medium"
```

## Swap Guidance

- Increase seriousness by reducing decorative motifs, using higher neutral contrast, and making labels more direct.
- Increase warmth by softening support color, increasing helper copy, and using friendlier empty states.
- Increase innovation by adding sharper motion, richer product diagrams, and clearer system feedback.
- Increase luxury by increasing whitespace, reducing visible metadata, and using fewer accent marks.
- Increase playfulness by adding subtle micro-interactions and brighter secondary accents.
- Increase editorial quality by widening text measures, improving article rhythm, and using larger media.
- Increase technical depth by strengthening code blocks, diagrams, tables, and structured navigation.
