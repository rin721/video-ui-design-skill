# Style Swap Prompt

Use this block to adjust brand expression while preserving the component system.

```text
Create a new visual direction using the same semantic component contract. Replace the primary brand scale, support accent scale, media tone, motion intensity, and typography emphasis while preserving accessibility, layout stability, responsive behavior, and component states.

Set brand dimensions:
- seriousness: [low | medium | high]
- warmth: [low | medium | high]
- innovation: [low | medium | high]
- luxury: [low | medium | high]
- playfulness: [low | medium | high]
- editorial: [low | medium | high]
- technical_depth: [low | medium | high]

Keep body text contrast at 4.5:1 or higher, large text at 3:1 or higher, and interactive UI boundaries at 3:1 where applicable. Use color with text, icon, or structure for every status. Do not reuse protected assets, brand marks, unique artwork, or distinctive page signatures.
```

## Adjustment Matrix

- Higher seriousness: quieter motion, stronger neutral contrast, fewer decorative marks.
- Higher warmth: softer support color, friendlier empty states, more human context.
- Higher innovation: clearer diagrams, sharper micro-interactions, stronger system feedback.
- Higher luxury: more whitespace, fewer metadata rows, richer media, fewer accents.
- Higher playfulness: brighter secondary accents, small animated affordances, lighter labels.
- Higher editorial: larger reading rhythm, stronger article hierarchy, wider media.
- Higher technical depth: stronger code blocks, data tables, sidebar hierarchy, and task cues.
