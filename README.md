# UI Design Skill Package

[English Skill](./eu)

该包提供一套原创、可复用的 UI 设计 skill，用于生成现代网站系统。内容包含语义化 token、组件规范、页面模板、视觉资产规则、prompt block 和 QA 清单。

## Contents

- `SKILL.md`：skill 主入口和执行契约。
- `design-system/`：设计原则、token、布局、排版、色彩、可访问性和品牌表达。
- `components/`：可复用组件 schema 与行为规则。
- `patterns/`：常见网站类型的可适配页面模板。
- `prompts/`：可单独使用的生成与审查 prompt block。
- `assets/`：视觉、图标和媒体替换指南。
- `qa/`：原创性、可访问性、响应式和清理清单。

## Ready For

- 网站生成。
- 风格替换。
- 组件生成。
- 页面模板生成。
- 品牌适配。
- 可访问性审查。

## Core Style

该系统偏向轻盈表面、紧凑工具导航、柔和强调、清晰内容层级、克制阴影，以及密集但易读的内容网格。所有规则都以可复用变量、模式和组件契约表达。
