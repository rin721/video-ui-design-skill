# 品牌视觉表达

## Brand Grammar

```yaml
brand_expression:
  personality:
    - "可接近"
    - "清新"
    - "有秩序"
    - "轻微玩味"
    - "可信"
  emotional_goal:
    - "让密集信息显得友好"
    - "降低发现和决策摩擦"
    - "邀请重复参与但不造成视觉疲劳"
  trust_strategy:
    - "清晰层级"
    - "可预测控件"
    - "易读 metadata"
    - "一致反馈"
  differentiation_strategy:
    - "柔和强调系统"
    - "紧凑导航"
    - "平衡内容密度"
    - "可替换视觉节奏"
  voice_visual_alignment:
    - "短而自信的标签"
    - "温和辅助文本"
    - "精确状态文案"
    - "直接行动提示"
  user_expectation:
    - "快速定位"
    - "快速扫描"
    - "低摩擦交互"
    - "安全错误恢复"
  brand_swap_parameters:
    seriousness: "medium"
    warmth: "medium"
    innovation: "medium"
    luxury: "low"
    playfulness: "medium"
    editorial: "medium"
    technical_depth: "medium"
```

## Swap Guidance

- 提高 seriousness：减少装饰符号，提高中性色对比，标签更直接。
- 提高 warmth：柔化辅助色，增加帮助文案，空状态更有人味。
- 提高 innovation：加入更清晰的图表、更利落的微交互和系统反馈。
- 提高 luxury：增加留白，减少可见 metadata，使用更少强调。
- 提高 playfulness：增加细微微交互和更明亮的辅助强调。
- 提高 editorial：扩大阅读节奏，增强文章层级，使用更大媒体。
- 提高 technical_depth：强化代码块、图表、表格和结构化导航。
